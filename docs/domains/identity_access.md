## Notes

- Snake case key: `identity_access`
- Generated by `php artisan create IdentityAccess`
- **RN-062 admin users:** CRUD without hard delete; deactivate with `is_active`; assign stores on create/update; block self lockout (`AUTH_CANNOT_MODIFY_SELF`); errors `AUTH_USER_NOT_FOUND`, `AUTH_EMAIL_DUPLICATE`.
- **Login CAPTCHA:** After `TURNSTILE_FAILURE_THRESHOLD` failed attempts (default 2), `POST /api/auth/login` requires `turnstile_token` verified via Cloudflare siteverify. Errors `AUTH_CAPTCHA_REQUIRED` / `AUTH_CAPTCHA_INVALID`; response `error.context.captcha_required` (+ `turnstile_site_key`). Disabled when `TURNSTILE_ENABLED=false` or secret empty.
